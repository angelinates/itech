Индивидуальное домашнее задание

Индивидуальное домашнее задание (ИДЗ) - один из видов работ в рамках курса Internet-технологии (1-й семестр).
Оценивается от 9 до 14 баллов - на основании защиты. При этом 9 баллов - минимальная оценка, будет автоматом выставлена при отработке всех лабораторных работ, но без выполнения собственно ИДЗ. Крайний срок сдачи ИДЗ - день выставления зачета.
На 22.11.2018 перечень тем находится на этапе формирования. До 6.12.2018 будут добавлены темы, посвященные написанию статических ресурсов и использованию языка PHP.
В данный момент доступны следующие темы заданий (указаны только ключевые требования, нюансы - обсуждаются на встречах в университете, или по почте):

Букмарклет для обработки страниц с вопросами/ответами сайта https://stackoverflow.com (пример обрабатываемой страницы - https://stackoverflow.com/questions/53223762/in-c-sharp-how-to-get-filenames-in-dropdown-without-the-full-path). Задача букмарклета - удаление ненужных пользователю элементов страницы (футер, левый сайдбар, лишние скрипты и т.п.) и стилистическое выделение принятого ответа, если он есть. Функционал букмарклета обсуждается.

Букмарклет для работы с ресурсом https://code.msdn.microsoft.com. Этот ресурс содержит примеры кода - на таких страницах, например, как https://code.msdn.microsoft.com/LINQ-Restriction-Operators-b15d29ca. Задача букмарклета - добавить к каждому блоку с кодом (класс scriptcode) кнопку для копирования содержимого блока в буфер обмена.

Букмарклет для обработки страниц веб-ресурса https://refactoring.guru/ru/refactoring. Задача букмарклета - удаление ненужных пользователю элементов страницы (сайдбар, кнопки соцсетей, лишние скрипты и т.п.). Функционал букмарклета обсуждается. Максимальная оценка - 13 баллов.

Букмарклет для стилизации значений прогноза температуры на сайте https://www.gismeteo.ua. Максимальная оценка - 12 баллов.
