2.2.2 Встраивание каскадных таблиц стилей в документ

    Существуют 3 метода:
  • inline-описание;
  • описание в секции заголовка;
  • внешний файл;

inline-описание: каждый тэг HTML имеет параметр style, в котором можно задать значения его свойств в соответствии с синтаксисом каскадных таблиц стилей:
<h1 style="color:red;">Заголовок отбражается шрифтом красного цвета</h1>

При внедрении таблицы стилей в документ правила задаются в стилевом блоке, ограниченном тэгами <style> и </style>, который должен размещаться в разделе <head>:
<style type="text/css"> body {background-color: #FF0000;} </style>

Таблицу стилей можно разместить в отдельном файле, подключаемом к веб-документу при помощи тэга <link>, задаваемого в разделе <head>:


<!-- Пример кода веб-документа -->
<html>
<head>
<title>Пример</title>
<link href="styles.css" type="text/css" rel="stylesheet">
</head>
<body> Пример хранения таблицы стилей в отдельном файле.</body>
</html>

/* Пример кода стилевого файла */ 
body { 
    background-image: url(background.png);
    background-position: left top;
	background-repeat: repeat-x;
}

Значение href задаёт путь к CSS-файлу, он может быть задан как относительно, так и абсолютно. Таким образом можно подключать таблицу стилей, которая находится на другом сайте.

В текущую стилевую таблицу можно импортировать содержимое CSS-файла с помощью команды @import. Этот метод допускается использовать при размещении таблиц в внешнем файле или в контейнере в секции заголовка. Общий синтаксис следующий:
@import url("имя файла") типы носителей;
@import "имя файла" типы носителей;
После ключевого слова @import указывается путь к стилевому файлу одним из двух приведенных способов - с помощью url или без него.
