Лабораторная работа №1 (2016)

Основы построения HTML-документов и подготовка изображений для Web
1.1 Цель работы
1. Изучить принципы построения HTML-документа, его структуру и форматирование. Рассмотреть возможности форматирования текста, организацию списковых структур, использование гиперссылок, форм и графики в WEB-документах. Понятие табличного представления данных.
2. Изучить практические приемы создания изображений для web-страниц: фона, баннеров.
3. Познакомиться с возможностями Github Pages.

1.3 Описание лабораторной установки
Для выполнения лабораторной работы рекомендуется использовать:
текстовый редактор Notepad++, имеющий подсветку синтаксиса;
графический редактор GIMP (GNU Image Manipulation Program).
Рекомендуемые средства являются свободным программным обеспечением. Для работы с хостингом GitHub Pages потребуется доступ к сети Интернет.
1.4 Порядок выполнения работы
  1. Создать HTML-документ на произвольную тематику на основе произвольного макета страницы (2-х и 3-х колоночного, например). Документ должен содержать блочные и строковые элементы, заголовки. Использовать табличную верстку.
      2. Согласно варианту, добавить в документ:
    • оформленный текст;
    • список;
    • таблицу;
    • форму с управляющими элементами;
    • гипертекстовые ссылки и изображения.
  2. Внести в проект указанные преподавателем изменения.
  3. Воспользоваться сервисом Github Pages (см. указания).
  4. Проверить отображение созданного веб-ресурса в различных браузерах.
  5. Проверить созданный документ на валидность с помощью пакета онлайн-сервисов, ПО CSE HTML Validator Lite или плагинов браузера.
Каждый этап задания рекомендуется делать на основе предыдущего. В целях лучшего освоения HTML, выполнять (пока!) без использования каскадных таблиц стилей.
Задания для лабораторной являются индивидуальными. Возможно совместное выполнение в бригаде вариантов всех ее участников.
Отработка выставляется после получения ссылки на страницу, выполненную в соответствии со всеми заданиями этого раздела.

1.6 Содержание отчета
Отчет по лабораторной работе должен содержать: цель работы, вариант задания, эскиз макета страницы, исходный код, изображение полученного документа, выводы по работе. Для защиты наличие распечатнного отчета не является необходимым.
