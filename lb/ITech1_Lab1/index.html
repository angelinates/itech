Лабораторная работа №1

Основы построения HTML-документов и подготовка изображений для Web
1.1 Цель работы
1. Изучить принципы построения HTML-документа, его структуру и форматирование. Рассмотреть возможности форматирования отдельных символов текста, организацию списковых структур, использование графики в WEB-документах. Понятие табличного представления данных.
2. Изучить практические приемы создания изображений для Web-страниц: фона, баннеров, кнопок.

1.3 Описание лабораторной установки
Для выполнения лабораторной работы рекомендуется использовать:
текстовый редактор Notepad++, имеющий подсветку синтаксиса;
графический редактор GIMP (GNU Image Manipulation Program).
Рекомендуемые средства являются свободным программным обеспечением.
1.4 Порядок выполнения работы
  1. Разработать макет сайта (пример эскиза макета) на произвольную тематику.
  2. Создать простой HTML-документ на основе разработанного макета страницы, содержащий элементы разметки текста, шрифтов, заголовков, логическое и физическое форматирование.
  3. Добавить в созданный документ элементы, определяющие списки, согласно варианта задания.
  4. Добавить элементы, создающие гипертекстовые ссылки, по варианту.
  5. Добавить элементы создания изображений, в соответствии с вариантом (см. примеры оформления фона документа, создания кнопок, создание баннера, можно анимированного.
  6. Добавить элементы создания таблиц, согласно варианта задания.
  7. Проверить отображение созданного документа в разных браузерах.
  8. Проверить созданный документ на валидность с помощью пакета CSE HTML Validator Lite или плагинов браузера.

Каждый этап задания рекомендуется делать на основе предыдущего. В целях лучшего освоения HTML, выполнять (пока!) без использования каскадных таблиц стилей.
Макет сайта может быть выполнен на любую тему. Это может быть справочная система к программному продукту, персональная домашняя страница и т.д.
Окончательный вариант задания демонстрируется в электронном виде. Возможно выполнение нестандартного задания. В этом случае тема задания должна быть согласована.

1.6 Содержание отчета
Отчет по лабораторной работе должен содержать: цель работы, вариант задания, разработанный эскиз макета страницы, исходный код, изображение полученного документа, выводы по работе.
