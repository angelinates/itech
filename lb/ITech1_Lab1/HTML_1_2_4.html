1.2.4 Создание таблиц в HTML

Каждая таблица должна начинаться тeгом <table> и завершаться тeгом </table>. Внутри этой пары тeгов располагается описание содержимого таблицы. Любая таблица состоит из одной или нескольких строк, в каждой из которых задаются данные для отдельных ячеек. Каждая строка начинается тeгом <tr> (Table Row) и завершается тeгом </tr>. Отдельная ячейка в строке обрамляется парой тeгов <td> и </td> (Table Data) или <th> и </th>(Table Header).

    Атрибуты таблицы (осуждаются спецификацией HTML5):
  • cellspacing = "10", расстояние между ячейками
  • cellpadding = "10", расстояние между границей ячейки и ее содержимым
  • width = "100%"
  • height = "50%"
  • align = "center| right| left", задает выравнивание таблицы относительно окна браузера
  • frame = "box| hsides| vsides| rhs| lhs| above| below| border", как отображать границы вокруг таблицы
  • rules = "groups| none| cols| rows| all"
  • bgcolor="#ffcc00"
    Параметры строки:
  • align = "left| right| center| justify" – выравнивание внутри строки;
  • valign = "top| bottom| middle"
  • bgcolor = "….."
  • height = "10%" – процентное соотношение строк для таблицы.
    Параметры ячейки:
  • width = "10%"(в процентах либо в пикселах);
  • align = "left| right| center| justify"
  • bgcolor = "..."
  • height = "10%"
  • rowspan = "2" – объдиняет по вертикали;
  • colspan = "2" - группирует по горизонтали.
